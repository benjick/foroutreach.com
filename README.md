# foroutreach.com [![Build Status](https://travis-ci.org/vegans/foroutreach.com.svg?branch=master)](https://travis-ci.org/vegans/foroutreach.com)

## Developing
Frontend
```
yarn
yarn start
```

Backend
```
docker-compose up
```

## Run in production

```
docker-compose -f docker-compose.yml -f docker-compose.prod.yml up -d
```
